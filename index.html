Реалізація стилів

**body**


**background-color: #FFFFFF;**: Встановлює білий колір фону для всього документа.
**margin: 0px;**: Видаляє стандартні поля навколо елементів, забезпечуючи прилягання до країв вікна браузера.

**menu_div**


**background-color: #2D2D2D ;**: Встановлює темний сірий колір фону для контейнера меню.
**height: auto;**: Автоматично підлаштовує висоту контейнера під його вміст.
**color: #FFFFFF;**: Встановлює білий колір тексту для елементів всередині контейнера.
**display: flex;**: Використовує модель flex-box для розміщення елементів всередині контейнера.
**place-items: center;**: Центрує елементи горизонтально і вертикально всередині контейнера.
**margin-left: auto; margin-right: auto;**: Центрує контейнер горизонтально на сторінці.

**tr_menu**


**margin-left: auto; margin-right: auto;**: Центрує елемент горизонтально.

**p_menu**


**color: #FFFFFF;**: Встановлює білий колір тексту.
**margin-left: auto; margin-right: auto;**: Центрує елемент горизонтально.

**a_menu**


**color: #FFFFFF;**: Встановлює білий колір тексту для посилання.
**font-size: 18px;**: Встановлює розмір шрифту 18 пікселів.
**font-family: 'Roboto', sans-serif;**: Використовує шрифт Roboto або будь-який інший шрифт без сірифів.
**background-image: linear-gradient(...)**: Створює лінійний градієнтний фон для тексту посилання.
**background-size: 200% 100%;**: Розтягує градієнт на 200% по ширині і 100% по висоті.
**background-position: -100%;**: Початкове положення градієнта зліва від тексту.
**display: inline-block;**: Використовує модель блочної розмітки для елемента, дозволяючи встановити ширину і висоту.
**padding: 5px 0;**: Встановлює відступи зверху і знизу.
**position: relative;**: Використовує позиціонування відносно батьківського елемента.
**-webkit-background-clip: text; -webkit-text-fill-color: transparent;**: Використовує CSS трюк для створення ефекту кольорового тексту на градієнтному тлі.
**transition: all 0.3s ease-in-out;**: Додає плавний перехід при зміні стану елемента.

**a:before**


**content: '';**: Додає псевдоелемент перед посиланням.
**background: #5d5e5d;**: Встановлює колір фону для псевдоелемента.
**display: block;**: Використовує блочну модель розмітки для псевдоелемента.
**position: absolute;**: Позиціонує псевдоелемент абсолютно відносно батьківського елемента.
**bottom: -3px; left: 0;**: Розміщує псевдоелемент під посиланням і зліва.
**width: 0; height: 3px;**: Спочатку псевдоелемент має нульову ширину і висоту 3 пікселі.
**transition: all 0.3s ease-in-out;**: Додає плавний перехід при зміні стану псевдоелемента.

**a:hover**


**background-position: 0;**: Зміщує градієнтний фон на початок тексту при наведенні курсору.

**a:hover::before**


**width: 100%;**: Розтягує псевдоелемент на всю ширину посилання при наведенні курсору.

**dropbtn**


**background-color: #2D2D2D;**: Встановлює темний сірий колір фону для кнопки розкривного меню.
**color: white;**: Встановлює білий колір тексту для кнопки.
**padding: 16px;**: Встановлює відступи всередині кнопки.
**font-size: 16px;**: Встановлює розмір шрифту для тексту кнопки.
**border: none;**: Видаляє рамку навколо кнопки.
**margin-left: auto; margin-right: auto;**: Центрує кнопку горизонтально.

**dropdown**


**position: relative;**: Використовує відносне позиціонування для розміщення розкривного меню.
**display: inline-block;**: Використовує блочну модель розмітки для елемента, дозволяючи встановити ширину і висоту.
**margin-left: 20px; margin-right: auto;**: Встановлює відступ зліва і центрує елемент горизонтально.

**dropdown-content**


**display: none;**: Спочатку приховує вміст розкривного меню.
**position: absolute;**: Використовує абсолютне позиціонування для розміщення розкривного меню.
**background-color: #2D2D2D;**: Встановлює темний сірий колір фону для вмісту розкривного меню.
**min-width: 160px;**: Встановлює мінімальну ширину вмісту розкривного меню.
**box-shadow: 0px 8px 16px 0px rgba(0,0,0,0.2);**: Додає тінь до вмісту розкривного меню.
**z-index: 1;**: Встановлює індекс z для вмісту розкривного меню, щоб він відображався поверх інших елементів.

**dropdown-content a**


**color: black;**: Встановлює чорний колір тексту для посилань в розкривному меню.
**padding: 12px 16px;**: Встановлює відступи всередині посилань.
**text-decoration: none;**: Видаляє підкреслення для посилань.
**display: block;**: Використовує блочну модель розмітки для посилань, щоб вони займали всю ширину розкривного меню.

**a_menu (друге використання)**


**margin: auto;**: Центрує елемент горизонтально і вертикально.
**text-align: center;**: Центрує текст всередині елемента.

**dropdown-content a:hover**


**background-color: #ddd;**: Змінює колір фону посилання при наведенні курсору.
